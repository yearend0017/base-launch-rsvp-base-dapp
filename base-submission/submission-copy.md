# Base Launch RSVP

## App Name

Base Launch RSVP

## Tagline

Reserve your Base seat

## Description

Reserve a launch event seat on Base and receive an onchain RSVP pass with live seat availability.

Character count: 97

## Category

Event / attendance / onchain utility

## Short Pitch

Base Launch RSVP turns event signups into a clean onchain action, giving each attendee a visible seat reservation and live capacity tracking.

## Production Environment Variables

- `NEXT_PUBLIC_LAUNCH_RSVP_CONTRACT_ADDRESS`: deployed RSVP contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
